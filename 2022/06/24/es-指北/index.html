3 es 集成

使用 spring data elasticsearch

Spring data elasticsearch 与 springboot、elasticsearch 版本对照表

es 的版本可以稍高，比如笔者用的是 springboot 2.4.x，但是 es 用的是 7.14.1

集成方式：

  1. 引入 maven 依赖

  2. 定义 repository

定义了 repository 之后，就可以直接通过 autowire 注解引入，然后进行 es 数据操作

4.1 时间存储与显示

使用 field 注解，格式化模式定义为：uuuu-MM-dd HH:mm:ss

这样，存储到 es 中的日期就是 年月日 时分秒 格式

设置为 UTC
